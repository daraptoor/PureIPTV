## üöÄ Getting Started

### Prerequisites
- Node.js (v18 or newer)
- npm or yarn
- Expo Go app on your mobile device (for development)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/daraptoor/PureIPTV.git
   cd PureIPTV
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npx expo start
   ```
